# Ryan Realty: Platform-Specific Publishing Runbook

## When to Use
Invoke this skill when Matt says "post this," "publish X," "share this to [platform]," or any request to publish content to Instagram, TikTok, Facebook, YouTube, LinkedIn, or Google Business Profile. This is the operational runbook that governs HOW content ships to each platform.

## Instagram Publishing Runbook

### Content Specifications

**Instagram Feed Post (Static)**
- Dimensions: 1080x1350 (4:5 ratio for property), or 1080x1080 (square for lifestyle/market content)
- File format: PNG or JPG
- Caption max: 2,200 characters (but write to 150 words for scanning)
- Hashtags: 5-10 niche hashtags, NO generic hashtag dumping (30 hashtags = algorithm penalty as of 2026)
- Hashtag placement: In caption body, not first comment
- First line: Must stop scroll (hook format per content-creation.md)

**Instagram Reels**
- Dimensions: 1080x1920 (9:16 vertical)
- Duration: 15-90 seconds of content (shorter = faster initial views = algorithm favor)
- Captions: Same 2,200 char limit; include DM-CTA
- Audio: Trending audio preferred (check CapCut library); original audio acceptable if strong
- Text overlay: Keep to 3-5 key phrases, readable at mobile size
- First 3 seconds: Must hook (visual + audio sync matters)
- Watermarks: Zero watermarks. Watermarked/cross-posted reels get downranked.

**Instagram Stories**
- Dimensions: 1080x1920 (9:16 vertical)
- Duration per frame: 3-5 seconds
- Sequence length: 3-7 frames per story set
- Text overlay: 5-7 words max per frame
- Interactive stickers: Swipe-up link (if available), poll, or quiz preferred; "DM me" sticker for CTAs
- Purpose: Real-time updates, property reveals, quick market takes, behind-the-scenes
- Story expiration: 24 hours (post, don't archive)

**Instagram Carousels**
- Slide count: 3-5 slides max (higher engagement on shorter carousels)
- Dimensions: 1080x1350 per slide
- Format: Property tours (before/after), market data breakdown, multi-part education
- Caption on Slide 1: Hook + DM-CTA
- Carousel layout rule: First slide stops scroll, subsequent slides answer the question
- Algorithm note: Carousels still underweight vs Reels as of April 2026; use Reels for max reach

**Instagram Trials Feature** (New as of 2024)
- What it is: Time-limited posts (24-48 hours) visible only to followers who opt in; high engagement signal
- Use case: Limited-time offers, property showings, flash community events
- Algorithm weight: Trials earn higher engagement ranking when enabled
- How to activate: Available via Instagram create menu; tag "Trial" when scheduling

**Instagram Collabs** (High-engagement feature)
- What it is: Invite local business / partner to co-author a post; appears on both profiles
- Use case: Partner with neighborhood businesses, local contractors, lifestyle brands
- Engagement signal: Collab posts earn 30%+ higher reach than solo posts
- Limitation: Both collaborators must have 10k+ followers OR collab must be invited by account with 10k+

**Instagram Broadcast Channels** (Audience segmentation)
- What it is: Send feed-style posts only to close followers or VIP audiences
- Use case: Exclusive property previews for buyer/seller networks, off-market listings
- Algorithm: Broadcast posts don't compete in the main feed; zero algorithmic disadvantage

### Publishing Workflow

**Before hitting publish:**
1. Caption: Run brand-voice:brand-voice-enforcement
2. Dimensions: Verify against specs above
3. DM-CTA: Confirm DM CTA is in caption (required for every Instagram post)
4. Hashtags: 5-10 niche only; check none are watermark-related or cross-promotion heavy
5. Link check: No external URLs in captions (native links only if Linktree is in bio)

**Publishing steps (via Chrome MCP browser):**
1. Navigate to instagram.com/accounts/login
2. Go to Create (+ icon)
3. Select format (Reel, Post, Story, Carousel, Trial, or Collab)
4. Upload asset(s) and caption
5. Set location tag (tag "Bend, Oregon" for local algorithm boost)
6. For Reels: confirm trending audio is used if applicable; add subtitle toggle if dialogue present
7. Schedule or publish immediately (Tuesday-Thursday 8am-4pm Bend time preferred)

**Post-publish checklist (within 30 minutes):**
- Pin a comment with the DM-CTA repeated (if caption is long or DM-CTA isn't visually prominent)
- For Reels: Set cover image (first frame or custom)
- Share Reel to Stories with link sticker
- Enable "Allow Sharing" in advanced settings
- Set reminders to respond to first comments within 1 hour (algorithm signal)

### Hashtag Strategy for Instagram
- Always mix: location (#BendOregon, #CentralOregon), property type (#AcreageForSale, #LuxuryRealEstate), lifestyle (#MountainLiving, #OutdoorLiving), market (#RealEstateTips, #HomeBuying), brand (#RyanRealty, #MattRyan)
- Avoid: #JustListed (low signal), #DreamHome (brand violation), #FeaturedListing (generic), hashtags with 1M+ posts (lost in noise)
- Never use: #FollowForFollow, #LikeForLike, engagement-bait hashtags (algorithm penalty)

## TikTok Publishing Runbook

### Content Specifications

**TikTok Video (Organic Feed)**
- Dimensions: 1080x1920 (9:16 vertical, portrait only)
- Duration: 60-90 seconds is sweet spot for watch-through (shorter = less income-driving algorithmic weight; longer than 2 min = niche only)
- File format: MP4 preferred
- Captions: Up to 2,200 characters; platform supports full captioning unlike Instagram
- Audio: Trending audio strongly preferred; algorithm gives 3-5x boost to trending sounds
- Watermarks: ZERO watermarks from other platforms (IG watermark, TikTok watermark from other creators, YouTube watermark = automatic downrank)

**On-Screen Text & SEO**
- Spoken keywords in first 3 seconds (TikTok's algorithm indexes spoken content)
- On-screen text with keywords helps with discovery (e.g., "Acreage For Sale Bend Oregon")
- Caption keywords matter for search (hashtag + text keywords in caption)
- Example: "Acreage homes in Bend Oregon | Investment land | Mountain views"

### Publishing Workflow

**Before publishing:**
1. Caption: Run brand-voice:brand-voice-enforcement
2. Dimensions: Confirm 1080x1920
3. Watermarks: Remove ALL watermarks (cross-platform watermarks tank TikTok reach)
4. Keywords: Verify spoken keywords in first 3s AND on-screen text
5. Trending audio: Check CapCut's trending library; use if relevant (not required but 3-5x boost)
6. Hashtags: 5-8 relevant + 1-2 trending (algorithm favors mixture)

**Publishing steps (via Chrome MCP browser):**
1. Navigate to tiktok.com, log in as Ryan Realty business account
2. Click "Upload" (+ icon or "Create")
3. Upload MP4 file
4. Add caption with keywords + hashtags
5. Add location tag ("Bend, Oregon" required)
6. Set cover image (first frame or custom)
7. Set who can view (Public recommended for algorithm reach)
8. Post immediately (no scheduling tool for organic; TikTok favors instant posts)

**Post-publish checklist (within 30 minutes):**
- Monitor first 30 seconds of view-through rate (if <40% by 30 min, consider deleting and reposting with stronger hook)
- Respond to ALL comments within 24 hours (algorithm signal; TikTok tracks response rate)
- Pin a helpful comment (not self-promotional) if one emerges early
- Share to TikTok Lite if available (doubles reach to older demographic)

### Algorithm Signals on TikTok (April 2026)
- Watch-through rate: King signal (completion matters most)
- Shares > Comments > Likes (very different from IG)
- Re-watches: Heavily weighted; rewatchable content performs 2x
- Recency: Videos continue earning views weeks/months later (unlike IG)
- Text-based search: TikTok is becoming search engine; optimize captions for discovery

## Facebook Publishing Runbook

### Content Specifications

**Facebook Page Reel (Primary format)**
- Dimensions: 1080x1920 (9:16 vertical) OR 1080x1080 (square)
- Duration: 15-90 seconds
- Captions: Full text acceptable (Facebook audience skews older, information-seeking)
- Algorithm: Reels outperform Feed posts 3-5x as of 2026
- Hashtags: 3-5 relevant (Facebook algorithm doesn't weight hashtags like IG/TikTok)

**Facebook Feed Post**
- Dimensions: 1080x1350 (4:5) or 1080x1080 (square)
- Captions: 150-300 words (longer-form acceptable on Facebook; audience expects context)
- Property addresses: Include full address (local search intent)
- Links: Native image post preferred; link posts see 30% lower organic reach
- Cross-posting from Instagram: No longer penalized as of 2025, but performance is lower; publish native when possible

**Facebook Strategy Notes**
- Suggested content to non-followers: 30%+ of total reach (even non-page-followers see Ryan Realty content)
- Video posts earn 3-5x reach vs static images
- Comments section: Engagement signal; respond to all comments within 1 hour
- Share button: Encourage shares (more valuable than likes for algorithmic reach)

### Publishing Workflow

**Before publishing:**
1. Caption: Run brand-voice:brand-voice-enforcement
2. Format: Prefer Reel over Feed post (3-5x better reach)
3. Dimensions: Confirm vertical (1080x1920) or square (1080x1080)
4. Call-to-action: "Message me" or "Call 541.213.6706" (DM-CTA less critical on FB; phone CTA acceptable)
5. Property details: Include address for local search boost

**Publishing steps (via Chrome MCP browser):**
1. Navigate to facebook.com, log in as Ryan Realty Page
2. Click "Create" (on page feed)
3. Select "Video" for Reels, "Photo/Video" for static
4. Upload file, add caption
5. Tag location (Bend, Oregon)
6. For properties: Include address + price
7. Publish immediately or schedule to Tuesday-Thursday 8am-4pm Bend time

**Post-publish checklist (within 30 minutes):**
- Pin post to top of page if it's a property or major announcement
- Respond to first 3-5 comments (engagement signal)
- Encourage sharing via comment: "Share this if you'd live here"
- Check Insights 24 hours later; note which posts outperform

### Cross-Platform Repurposing (Facebook + Instagram)
- Native posts on BOTH platforms outperform sharing Instagram to Facebook
- Slight caption adjustments for audience (Facebook = older, detail-seeking; IG = younger, visual-focused)
- Use same video/image asset, but publish separately with platform-native captions

## YouTube Publishing Runbook

### YouTube Shorts (60 seconds max)

**Specifications**
- Dimensions: 1080x1920 (9:16 vertical)
- Duration: 15-60 seconds (Shorts algorithm favors <45 sec)
- Format: Vertical video only
- Title: MUST be a searchable query (e.g., "Acreage Homes Under $1.5M in Bend Oregon" not "Check this out")
- Description: 150 words with keywords; include links to website or listing

**Publishing workflow:**
1. Navigate to youtube.com, log in
2. Click "Create" (camera icon) > "Upload Short"
3. Upload vertical MP4
4. Title as keyword query (affects YouTube search ranking)
5. Add description with relevant keywords + link to ryan-realty.com
6. Add hashtags (YouTube allows up to 15)
7. Publish

### YouTube Long-Form (Series Format)

**Specifications**
- Dimensions: 16:9 landscape (1920x1080) or 9:16 vertical
- Duration: 5-15 minutes optimal for watchtime-based monetization
- Title: Keyword-first (e.g., "Buying a Home in Bend Oregon: 5 Steps You Must Know")
- Thumbnail: Custom thumbnail required (1280x720); must include hook text
- Description: 200-300 words; includes links to website, call-to-action

**Content ideas:**
- "How to Buy a Home in Bend" (buyer education series)
- "Market Data Breakdown: Bend Real Estate Trends" (monthly market commentary)
- "Neighborhood Deep-Dive: [Neighborhood Name]" (Snowdrift Visuals footage)
- "Why I Left [Franchise] for Independent Brokerage" (personality/story)

**Publishing workflow:**
1. Navigate to youtube.com, log in as Ryan Realty channel
2. Click "Create" > "Upload Video"
3. Upload MP4
4. Title (keyword-first), description (200+ words), add thumbnail
5. Add hashtags + Keyword Tags (hidden but search-weighted)
6. Set visibility (Public for maximum reach)
7. Enable comments, set community tab if available
8. Publish

### YouTube Algorithm Notes
- Title click-through rate (CTR) is primary driver (optimize title for search keywords)
- Watch-time matters more than views (longer videos = better watch-time signal if good)
- Subscriber growth from Long-Form feeds Shorts algorithm
- Shorts serve to non-subscribers (broader reach than Feed posts on IG)

## TikTok API Publishing (Built In)

**Status (April 2026):** Direct API integration available via `/api/social/publish-tiktok` endpoint.

### Available Routes

**POST /api/tiktok/authorize**
- Generates OAuth2 consent URL and redirects user to TikTok login
- Stores CSRF state in Upstash Redis (10-minute TTL)
- No parameters required
- Redirect destination: TikTok consent screen

**GET /api/tiktok/callback**
- OAuth2 callback handler for TikTok redirect
- Validates state parameter against Redis
- Exchanges auth code for access/refresh tokens
- Stores tokens in Supabase table `tiktok_auth` (row id: 'default')
- Query parameters: code, state, error, error_description
- Success redirect: `/admin/social?tiktok=connected`
- Error redirect: `/admin/social?tiktok=error&reason=[message]`

**POST /api/social/publish-tiktok**
- Publish video to TikTok using stored credentials
- Requires header: `X-Cron-Secret: [CRON_SECRET]`
- Request body:
  ```json
  {
    "videoUrl": "https://...",
    "title": "Video title",
    "privacyLevel": "PUBLIC"
  }
  ```
- Auto-refreshes expired access tokens via refresh_token
- Response: `{ success: true, publishId: "...", status: "submitted" }`
- Supports public URLs only (PULL_FROM_URL method)

### Integration Flow

1. **Initial Auth (One-time):**
   - Navigate user to `/api/tiktok/authorize`
   - User logs in to TikTok and grants permissions
   - Redirected back to `/api/tiktok/callback` with code
   - Tokens stored in `tiktok_auth` table

2. **Publishing (Repeatable):**
   - Call `POST /api/social/publish-tiktok`
   - Pass video URL + title
   - API auto-refreshes tokens if expired
   - Returns publish_id on success

### Database Schema (Comment: Create if needed)

```sql

  id TEXT PRIMARY KEY,
```

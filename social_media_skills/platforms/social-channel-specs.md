## Instagram

### Profile Setup
- **Account type:** Business (not Creator). Reason: full scheduling API access, contact buttons, lead forms, action buttons (Book). Trade-off: commercial music library only, but workarounds exist.
- **Avatar:** Upload 1080×1080 px; stored at 320×320; displayed as circle at 110×110 px (mobile) / 180×180 px (desktop). JPG or PNG. Max 30 MB. Keep face/logo centered — corners are clipped.
- **Banner/Cover:** N/A — Instagram has no banner field.
- **Name/Display Name:** 30 characters. Searchable in-app — pack with keywords (e.g., "Matt Ryan | Bend Realtor").
- **Username/Handle:** Max 30 characters. Letters, numbers, periods, underscores only. No hyphens. No frequency limit on changes.
- **Bio:** 150 characters hard limit. Line breaks supported. Emojis supported (count as 1–2 chars). Hashtags tappable. URLs typed in bio are NOT clickable — use the Link field.
- **Link field:** Up to 5 native links (no follower threshold). All account types. Add via Edit Profile → Links.
- **Contact fields:** Call, Text, Email, Directions buttons — Business accounts only. Add via Edit Profile → Contact Options.
- **Category/Label:** "Real Estate Agent" — shown below name field, toggleable. Business accounts only.
- **CTA button:** "Book" (connect Calendly/Acuity) is highest-value for RE consultation bookings.

### Features to Enable
- Contact buttons (Call + Email) — direct lead contact from profile
- Action button "Book" → Calendly — removes friction from consultation requests
- Threads badge → show only if Threads account is active
- Broadcast Channel — create for opted-in market update subscribers; one-way push channel
- Close Friends list — use for VIP buyer alerts, pre-market listings
- 3 Pinned Posts — slot 1: intro/brand; slot 2: best market content; slot 3: social proof
- Story Highlights (5–7 covers) — titles ≤10 chars: Start Here, Buyers, Sellers, Bend Life, Market, Reviews, Process

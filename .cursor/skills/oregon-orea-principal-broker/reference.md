## ORS hooks (statutes)

Principal brokers should know where **ORS ch. 696** lives for big-picture duties; **specific filing and supervision mechanics** are often spelled out in **OAR 863**. Use the legislature’s ORS pages for current statutory text.

## OREF vs OREA

- **OREF** = Oregon Real Estate Forms, LLC (private standardized forms). See `oregon-real-estate-oref` skill and **https://orefonline.com/**.
- **OREA** = Oregon **Real Estate Agency** (state regulator). Forms compliance and **regulatory** compliance are related but **not identical**; both must be satisfied in practice.

## Training discipline for agents using this repo

1. Read **OREA** “Laws and rules” and the **863** chapter index at least once per renewal cycle for orientation.
2. For any automated file audit (e.g. SkySlope JSON + PDFs), re-run findings against **OAR 863-015-0250**-style completeness and **863-015-0140**-style supervision themes—not only filenames.
3. Keep **brokerage written policies** (PB review initials, naming, EMD handling, who may close) as the **local** source of truth when they are stricter than generic checklists.
